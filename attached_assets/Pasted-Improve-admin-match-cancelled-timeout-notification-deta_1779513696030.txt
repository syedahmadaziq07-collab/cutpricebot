RESPONSIBLE USER RULES
━━━━━━━━━━━━━━━━━━

1. If one user submitted proof and the other did not approve/reject:
Responsible User = the user who did not approve/reject.

2. If one user pressed Done Cut but did not upload proof:
Responsible User = that user.

3. If one user manually cancelled:
Responsible User = the user who cancelled.

4. If both users did nothing:
Responsible User = BOTH USERS.

5. If system cannot detect clearly:
Responsible User = Unknown / needs manual review.
