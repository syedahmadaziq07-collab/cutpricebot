Add a permanent TikTok username lock system.

━━━━━━━━━━━━━━━━━━
NEW RULE
━━━━━━━━━━━━━━━━━━

1 Telegram account can only register 1 TikTok username forever.

Once a user registers a TikTok username:
- lock that username to their Telegram account
- user cannot change to another TikTok username
- /start must not ask them to register again
- sending a different TikTok profile link must be rejected

━━━━━━━━━━━━━━━━━━
DUPLICATE USERNAME RULE
━━━━━━━━━━━━━━━━━━

TikTok username must be unique.

If another Telegram account tries to register a TikTok username already used by someone else, reject it.

Message:

Sorry 😵

This TikTok username is already taken by another user.

━━━━━━━━━━━━━━━━━━
EXISTING DUPLICATE CLEANUP
━━━━━━━━━━━━━━━━━━

Before enabling the new lock system:

1. Scan database for duplicate TikTok usernames.
2. Keep ONLY the oldest/original owner of that username.
3. All newer duplicate users must:
   - have their TikTok username removed/reset
   - be forced to register a different TikTok username again

Reset affected users:
- tiktokUsername = null
- tiktokUsernameLocked = false

When affected user opens /start:

⚠️ Your previous TikTok username was removed because it conflicted with another account.

Please register a new TikTok username to continue ✨

━━━━━━━━━━━━━━━━━━
